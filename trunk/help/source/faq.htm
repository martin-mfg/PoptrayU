Frequently Asked Questions
Q1. Does PopTrayU support webmail (Hotmail,Yahoo,etc.)?
A1. Not by default, unless your webmail provider has a POP3 interface available. However, PopTrayU is compatible with PopTray plugins, so some webmail plugins exist.

Q2. The reply/new buttons don't open the specified e-mail program.
A2. This is how the program was designed. Since it supports any e-mail client, it cannot know how to specify a reply by just knowing the executable file. Therefore it uses either a "mailto:" link, or Simple-MAPI (depending on the option), which will open your default e-mail client.

Q3. Can I setup PoptrayU to run in a multi-user environment?
A3. Yes. PopTrayU saves all its settings in the PopTray.ini file. You can specify a different directory for the file on the command-line. By using the command-line
"C:\Program Files\PopTrayU\PopTrayU.exe" "%USERPROFILE%"
or "C:\Program Files\PopTrayU\PopTrayU.exe" "%HOMEDRIVE%%HOMEPATH%"
your PopTray.ini file will be created in your user home directory, which is different for each user on the PC.
